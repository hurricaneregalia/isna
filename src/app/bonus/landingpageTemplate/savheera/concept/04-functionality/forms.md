# Forms: Savheera Jewelry Landing Page

## Tujuan File

File ini menjelaskan panduan desain form (contact form, newsletter signup, inquiry form) untuk landing page.  
AI agent akan menggunakan panduan ini untuk membuat form `.js` dengan konsistensi visual dan UX sesuai brand Savheera.

## Panduan Form

1. **UX & Interaction**

   - Validasi input sederhana
   - Feedback lembut (success/error) dengan warna brand soft
   - Fokus pada pengalaman lembut dan elegan

2. **Accessibility**
   - Label jelas
   - Kontras teks terhadap background
   - Tab order logis untuk navigasi keyboard
