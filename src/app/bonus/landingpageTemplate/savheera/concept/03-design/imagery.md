# Imagery: Savheera Landing Page

## Tujuan File

File ini menjelaskan panduan penggunaan gambar dan visual untuk landing page.  
AI agent akan menggunakan ini untuk menempatkan hero image, ilustrasi, dan visual lain secara konsisten dan elegan, sesuai karakter brand Savheera: elegan, aspiratif, dan berkelas.

---

## Sumber Image Gratis (Untuk Development)

**WAJIB:** Gunakan URL image gratis dari website berikut untuk development:

### Recommended Sources:

1. **Pexels.com** - High-quality stock photos gratis

   - URL format: `https://images.pexels.com/photos/[photo-id]/[description].jpg`
   - Contoh: `https://images.pexels.com/photos/761293/pexels-photo-761293.jpeg`

2. **Unsplash.com** - Professional photography gratis

   - URL format: `https://images.unsplash.com/photo-[photo-id]?w=800&auto=format&fit=crop`
   - Contoh: `https://images.unsplash.com/photo-1596944254735-2a1b5c3d5f3a?w=800&auto=format&fit=crop`

3. **Pixabay.com** - Free images and illustrations
   - URL format: `https://pixabay.com/photos/[photo-id]/[description].jpg`

## Panduan Gambar

### 1. Hero Image

- Gambar utama di Hero Section
- Tema: perhiasan untuk momen spesial (wedding, gala, anniversary)
- Layout: full-width
- Fokus pada kilau, detail, dan kesan elegan
- Warna dan tone konsisten dengan palet Savheera (Ivory, Champagne, Soft Rose, Gold, Deep Navy)

### 2. Ilustrasi Section

- **Problem Section:** ilustrasi situasi sulit saat memilih perhiasan
- **Solution / Feature Section:** ikon atau gambar produk eksklusif
- **Proof Section:** foto pelanggan dengan perhiasan di momen spesial
- Gunakan gambar profesional dan high-quality
- Hindari background yang mengganggu fokus
- layout `lg:w-10/12 mx-auto w-full`

### 3. Style & Konsistensi

- Semua gambar harus **high-quality, profesional, dan konsisten** dengan tone brand
- Spacing antar visual: `py-16` atau `py-20`
- Gunakan white space untuk menjaga elegansi dan keterbacaan
- Hindari gambar yang terlalu ramai atau tidak relevan dengan tema momen spesial

### 4. Ikon

- Gunakan satu gaya konsisten (outline atau solid)
- Ukuran seragam di seluruh halaman
- Warna mengikuti **primary** atau **accent** sesuai konteks
- Ikon berfungsi sebagai pendukung visual, bukan fokus utama

---

## Catatan untuk AI Agent

- Gunakan panduan ini sebelum membuat file section `.js`
- Pastikan semua visual **menunjang tone elegan, soft selling, dan aspiratif**
- Visual harus mendukung storytelling dan menonjolkan momen spesial
- Selalu jaga konsistensi warna, spacing, dan kualitas gambar
