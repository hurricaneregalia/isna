## Konsep Dasar

Menampilkan grid card perhiasan Savheera dengan pendekatan elegan dan inspiratif. Setiap card menampilkan foto editorial perhiasan, nama koleksi, harga yang transparan, dan ajakan lembut untuk menemukan pilihan sempurna momen istimewa.

**Target file implementasi (akan dibuat):**

- `src/app/bonus/landingpageTemplate/savheera/sections/SavheeraProductListSection.js`

## Filosofi Desain

### 1. Brand Alignment

- **Elegan & Minimalis**: Desain bersih yang fokus pada keindahan perhiasan
- **Aspiratif**: Menampilkan perhiasan dalam konteks momen spesial
- **Soft Selling**: Menginspirasi tanpa tekanan komersial

### 2. Tone & Voice

- Lembut dan menginspirasi
- Fokus pada makna dan emosi
- Kata kunci: "memukau", "istimewa", "elegan", "pilihan sempurna"

## Struktur Visual

### 1. Product Card Layout

- **Header Section**:
  - Badge eksklusif ("Limited Edition", "New Arrival") dengan warna Gold
  - Subtle indicator untuk koleksi spesial
- **Image Section**:
  - Foto editorial perhiasan dengan aspect ratio 4:5 (portrait)
  - Hover effect dengan soft zoom dan subtle glow
  - Background Ivory/Champagne untuk kesan mewah
- **Content Section**:
  - Nama koleksi dengan font Serif elegan (max 2 lines)
  - Deskripsi singkat inspiratif ("Untuk momen berkesan Anda")
  - Harga dengan format IDR yang elegan
  - Material information (18K Gold, Diamond, etc.)
- **Action Section**:
  - CTA lembut: "Jelajahi Koleksi" (warna Gold)
  - Secondary: "Lihat Detail" (warna Soft Rose)

### 2. Grid Layout

- **Desktop**: 3 columns grid (lebih bernapas)
- **Tablet**: 2 columns grid
- **Mobile**: 1 column grid
- Gap antar cards: 2rem (white space yang cukup)
- Container padding: py-24

### 3. Responsive Behavior

- Card height konsisten dengan white space yang proporsional
- Typography yang tetap elegan di semua viewport
- Touch-friendly buttons dengan size yang sesuai
- Image optimization untuk mobile performance

## Styling Guidelines (Savheera Brand)

### Tailwind CSS + DaisyUI Implementation

- **Theme**: Gunakan theme "savheera" yang telah didefinisikan
- **Color System**: 60% netral (Ivory/Champagne), 30% sekunder (Soft Rose/Navy), 10% aksen (Gold)
- **Border Radius**: Hanya pada button, card, dan image container
- **Spacing**: Gunakan spacing konsisten (kelipatan 4 atau 8)

### Color Scheme (DaisyUI Classes)

- **Primary**: `bg-primary` (Gold #D4AF37) untuk CTA dan highlights
- **Secondary**: `bg-secondary` (Soft Rose #F4C2C2) untuk secondary elements
- **Neutral**: `bg-base-100` (Ivory #FFFFF0) untuk background cards
- **Base**: `bg-base-200` (Champagne #F7E7CE) untuk container
- **Text**: `text-base-content` (Deep Navy #1B1F3B) untuk readability

### Typography

- **Heading**: Font Serif (Playfair Display) untuk nama koleksi
- **Body**: Font Sans-serif (Lato) untuk deskripsi dan UI
- **Price**: Lato dengan weight yang konsisten
- **Buttons**: Lato medium dengan letter spacing yang elegan

### Spacing (Tailwind CSS)

- **Card Padding**: `p-8` (lebih bernapas)
- **Image Margin**: `mb-6`
- **Button Gap**: `gap-4`
- **Section Margin**: `mb-12`
- **Container Padding**: `py-24`
- **Grid Gap**: `gap-6`

### Animation & Interaction

- **Hover**: Gunakan kelas default Tailwind (`hover:scale-105`, `transition-transform`)
- **Fade-in**: Gunakan AOS library untuk elemen yang muncul di viewport
- **Transitions**: Smooth transitions pada semua interactive elements

### Layout Structure

- **Grid**: Gunakan `grid` atau `flex` untuk alignment yang rapi
- **White Space**: Cukup spacing agar layout terasa ringan dan elegan
- **Balance**: Jaga keseimbangan antara teks dan visual

### Data Management

- **Source**: Semua data dari `SavheeraDatabase.js`
- **Images**: URL gambar disimpan di database
- **Icons**: React Icons dari database
- **Text**: Semua teks dinamis dari database
